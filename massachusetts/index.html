How Massachusetts Child Support Is Calculated

  1. Massachusetts uses WEEKLY gross income and a formula called Table A -- a piecewise (not flat) function of combined weekly income that gives the basic support amount for ONE child.
  2. Add both parents' weekly gross income to get combined available income, then apply Table A's formula to get the 1-child base amount.
  3. Multiply the 1-child amount by a fixed factor for the actual number of children: 1.40 for two, 1.68 for three, 1.85 for four, 1.94 for five (six or more reuses the five-children factor).
  4. Add weekly childcare and health insurance costs to get the total combined support amount, then each parent pays their share in proportion to their income.
  5. If the paying parent's OWN weekly income (not combined) is $391 or less, their obligation is instead calculated from Table A using their own income alone -- this is Massachusetts's Self-Support Reserve, and no add-ons apply in this zone.
  6. Above $8,654/week (about $450,000/year) combined available income, the result is only a presumptive MINIMUM -- courts have discretion to order more.
  7. This models the standard case only. It does not model the Parenting Time proportional adjustment for substantial shared parenting time, the Table C age adjustment for children 18 or older, or deductions for other children/prior support.

Massachusetts Child Support Worksheet Walkthrough

Example: Parent A earns $1,200/week gross, Parent B earns $800/week gross (combined $2,000/week). There are two children, who live primarily with Parent B.

  1. Combined weekly income: $1,200 + $800 = $2,000.
  2. Table A base for 1 child at $2,000: $346 + 18% of ($2,000 - $1,600) = $346 + $72 = $418.
  3. Multiply by the two-children factor: $418 x 1.40 = $585 (rounded).
  4. Parent A's income share: $1,200 / $2,000 = 60%.
  5. Parent A pays: $585 x 60% = $351/week.

Does Massachusetts use a schedule table or a formula for child support?

A formula. Unlike most states' printed schedule tables, Massachusetts's Table A is a piecewise mathematical function of combined weekly income (different linear formulas apply across 8 income ranges) -- this calculator uses that exact formula directly, transcribed from the official worksheet's own calculation logic.

What is Massachusetts's Self-Support Reserve?

If the paying parent's own weekly gross income is $391 or less, Massachusetts calculates their obligation from Table A using their own income alone, regardless of the other parent's income, with no childcare or health insurance add-ons.

How does shared parenting time affect Massachusetts child support?

Massachusetts has a separate Parenting Time proportional adjustment (Worksheet Section 6) that can cap or adjust support when parenting time is substantially shared, comparing each parent's share of parenting time against their share of the support obligation. This calculator does not model that adjustment -- it estimates the standard case where one parent has primary care.
